Q2
The CSS _______ property defines the font to be used for an HTML element.

Ans: The CSS font-family property defines the font to be used for an HTML element. < h1 style="font-family:comic-sans;" => font will be in comic sans.

Q3
The CSS _______ property defines the text size for an HTML element.

Ans: The CSS text-property defines the text size for an HTML element. < h1 style="text-size:10px;" > => text will be 10px.

Q4
The CSS _______ property defines the horizontal text alignment for an HTML element.

Ans: The CSS text-align property defines the horizontal text alignment for an HTML element. < h2 style="text-align:center;" > => the text will be in the centre of the page.
