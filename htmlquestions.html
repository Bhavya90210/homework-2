a)- What is < !doctype > in html page and what is the significance of it in a web page?

Ans: < !doctype > is important in an html page because it lets the browser know what type of document you are doing, for example < !doctype html > => this will let the browser know that the document type is html.

b) How can you view the HTML source code of a web page>

Ans: To view the soruce code of a web page, right click on the browser and then click 'view page source' and it will be there.
